Portfólio — Ciência da Computação

Repositório com projetos desenvolvidos durante o curso CC50 — versão em português do CS50 de Harvard, oferecido pela Fundação Estudar.


📚 Sobre o Curso
O CC50 é o curso de Introdução à Ciência da Computação de Harvard, traduzido e oferecido gratuitamente pela Fundação Estudar no Brasil. O curso aborda fundamentos de programação, algoritmos, estruturas de dados e desenvolvimento de software.
